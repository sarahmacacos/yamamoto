HTML

o que e?

HTML e a linguagem que estrutura tudo que aparece na pagina. texto, imagem, botao... tudo começa com HTML.

tags semanticas

desde o HTML5 existem tags que descrevem o que o conteudo significa, nao so como ele aparece.

  • <header> cabecalho
  • <nav> navegacao
  • <main> conteudo principal
  • <section> secao
  • <article> artigo
  • <aside> conteudo extra
  • <footer> rodape

estrutura basica

<!DOCTYPE html>
<html lang="pt-BR">
  <head>
    <meta charset="UTF-8">
    <title>meu site</title>
  </head>
  <body>
    <h1>ola mundo!</h1>
  </body>
</html>

tags mais usadas

  • <h1> titulo
  • <p> paragrafo
  • <a> link
  • <img> imagem
  • <div> divisao

CSS

o que e?

CSS cuida da aparencia da pagina. cor, fonte, tamanho, layout... sem CSS o site e so texto feio.

como escrever

h1 {
  color: yellow;
  font-size: 2rem;
}

.caixa {
  background: #1e3a5f;
  padding: 10px;
  border-radius: 8px;
}

flexbox

a forma mais facil de organizar elementos lado a lado ou empilhados.

.container {
  display: flex;
  justify-content: center;
  gap: 20px;
}

responsividade

pra o site funcionar no celular a gente usa media query.

@media (max-width: 600px) {
  .menu {
    flex-direction: column;
  }
}

JavaScript

o que e?

JavaScript e a unica linguagem de programacao que roda no navegador. com ela da pra deixar o site interativo de verdade.

variaveis e funcoes

let nome = "Ana";
const idade = 17;

function oi(nome) {
  return `ola, ${nome}!`;
}

const dobrar = n => n * 2;

eventos e DOM

o JS consegue pegar qualquer elemento da pagina e alterar ele quando o usuario faz alguma coisa.

document.querySelector("#btn")
  .addEventListener("click", () => {
    alert("clicou!");
  });
